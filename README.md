3. انحراف معیار (Standard Debiation):

- انحراف معیار توزیع نمایی نیز به صورت زیر است:

$$\sigma=\frac{1}{\lambda}$$


توزیع نمایی دارای پارامتری به نام λ (نرخ یا rate) است که بیانگر نرخ رخداد رویدادها است. در واقع، λ = 1/scale است.
